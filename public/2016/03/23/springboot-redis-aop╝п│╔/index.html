###　注入目标

说明如下：

  • && args(user_id) 这个是目标函数的参数，只需要参数名，不需要参数类型，参数类型在后面指定
  • (ProceedingJoinPoint joinPoint,int user_id)　如果有目标函数的参数，则需要在joinPoint后面加入参数类型和参数
  • Object user 使用User user不起作用，不知道为什么。可能是User是特殊的类，和系统类冲突
